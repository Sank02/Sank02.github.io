# Sanket Bhosale — Data Analyst Portfolio
 
A personal portfolio website built as a single-page application with a clean, editorial minimalist design.
 
**Live site:** `https://sank02.github.io`
 
---
 
## Overview
 
This portfolio showcases my transition from Software Engineering into Data Analytics. It is designed to communicate both technical depth and business-facing analytical thinking, targeting Data Analyst, Business Analyst, and Analytics Engineer roles.
 
---
 
## Sections
 
| Section | Description |
|---|---|
| **Hero** | Name, animated role title (typed effect), tagline, and resume download |
| **About** | Background, career pivot story, profile photo, and key stats |
| **Skills & Certifications** | Tool stack grouped by category, with linked certification badges |
| **Projects** | 5 featured end-to-end data projects with GitHub links |
| **Blogs** | 3 technical articles (hosted on Notion) |
| **Experience & Education** | Timeline covering SE role, Forage simulations, and BE degree |
| **Contact** | Email, LinkedIn, and GitHub links |
 
---
 
## Tech Stack
 
| Layer | Details |
|---|---|
| **Structure** | Semantic HTML5 |
| **Styling** | Vanilla CSS with custom properties (CSS variables) |
| **Scripting** | Vanilla JavaScript - no frameworks or dependencies |
| **Fonts** | Google Fonts - Bebas Neue, DM Serif Display, DM Sans |
| **Icons** | Inline SVG |
| **Hosting** | GitHub Pages |
